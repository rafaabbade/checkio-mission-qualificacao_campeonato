Posso colocar um if dentro de outro?

Sim, você pode colocar um if dentro de outro. Isso é chamado de "if aninhado". Apenas certifique-se de manter a indentação correta. A indentação é o espaço no início de uma linha de código que mostra que ela está dentro de um bloco, como um if ou else.

Deve ter um jeito mais fácil de fazer isso, certo?

Sim, você pode combinar as duas condições usando o operador and. Por exemplo: if idade >= 14 and competicoes >= 5:. Isso será abordado em mais detalhes nos exercícios mais avançados.
