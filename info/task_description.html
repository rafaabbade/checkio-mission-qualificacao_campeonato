Enunciado

Helena precisa de um sistema para verificar se os atletas estão qualificados para participar com base em sua idade e no número de competições em que já participaram. Caso o atleta tenha 14 anos ou mais e tenha participado de pelo menos 5 competições, ele está qualificado.

Entrada: Um inteiro para a idade e outro inteiro para o número de competições.

Saída: Uma string "Qualificado para o campeonato" ou "Não qualificado para o campeonato".
